C# Web Application Project

This repository highlights my work on a C# project using MVC and Entity Framework to add functions to an existing website for a theatre company. The focus was to expand their blog section to allow comments, subcomments, image uploads, and a role authorization to moderate those comments. The work involed creating models, adding databases, writing custom logic for the CRUD functionality, and styling pages. In addition to C#, several other languages and libraries were used, such as: JavaScript, AJAX, SQL, JSON, jQuery, and Bootstrap.

Python Web App Project

This website was created using the django framework. Its theme is that of an online 'game cubbard'. It lets users create and update records of the games they own, add and remove items from a wishlist, read reviews generated by web scraping with beautifulsoup, and search API results based on user input.

Angular Web Application
